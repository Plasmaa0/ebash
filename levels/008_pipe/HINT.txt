Чтобы прочитать файл, используйте команду `cat filename`, где `filename` - имя файла, который нужно прочитать.

Чтобы отфильтровать строки со словом `apple`, используйте команду `grep "apple"`.

Чтобы объединить команды в конвейер, используйте символ "|".

Должно получиться что-то такое:
команда1 | команда2 
В таком случае команда2 получит на вход вывод команды 1.

По аналогии с данным примером подайте на вход команды grep с заданным фильтром "apple" вывод команды cat, которая прочитает файл.

А вывод любой команды можно перенаправить из консоли в файл с помощью оператора >
Напрмер:
cat fruits.txt > more_fruits.txt - создаст файл more_fruits.txt, в который запишет результат вывода команды 'cat fruits.txt'
