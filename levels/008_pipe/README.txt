Прочитать файл `fruits.txt` отфильтровать из него строки содержащие слово `apple` и записать их в файл `apples.txt`

Добро пожаловать в уровень для обучения команде `cat`, `grep` и оператору конвейера!

Этот уровень поможет вам научиться читать файлы, фильтровать строки по определенному условию и использовать конвейер для передачи вывода одной команды в другую.

Первая команда, которую мы будем использовать - это команда `cat`. Она используется для вывода содержимого файла в терминал. Данная команда принимает лишь один аргумент - имя файла.

Вторая команда - это команда `grep`. Она используется для поиска строк, соответствующих заданному шаблону. Синтаксис команды выглядит так: `grep шаблон имя_файла`. Например, если мы хотим найти все строки с словом "apple" в файле "fruits.txt", мы можем использовать команду "grep apple fruits.txt".

Третья команда - это оператор конвейера "|". Он используется для передачи вывода одной команды в другую. Попробуйте сначала прочитать файл с помощью команды `cat` а потом с помощью оператора конвейера `|` передать содержимое файла на фильтрацию команде `grep`. 

Чтобы успешно выполнить этот уровень, вам нужно будет использовать все три команды вместе, чтобы прочитать файл, отфильтровать только строки со словом "apple" и записать результаты в файл "apples.txt".
